# CSC-210-Spring2020
Contains all the code for CSc 210: Introduction to Software Development. Taken in Spring 2020. 

Language Used: JAVA

Key concepts learnt:
- Iterative Development
- Inheritance
- Polymorphism
- Object Oriented Programming (Classes and Objects)
- Recursion
- Recursive Backtracking
- Time Complexity
- Improving efficiency
- Heuristic Algorithms
- Data Structures:
  * Stacks and Queues
  * Lists (ArrayLists and Linked lists)
  * Sets (HashSets and TreeSets)
  * Priority Queues
  * Binary Heaps (Min and Max heaps)
